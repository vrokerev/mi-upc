# mi_upc_x

TIU Virtual clone for the university ID screen with editable profiles, moving
clouds, and a right-side edit drawer.

## Run

1. `flutter pub get`
2. `flutter run`

## Notes

- Use the edit drawer to update name, major, student code, banner ID, campus,
  and photo.
- Open the edit drawer by swiping with two fingers from right to left.
